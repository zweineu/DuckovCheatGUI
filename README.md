## 🎮 使用方法

### 打开/关闭菜单
- 按 **Home** 键打开或关闭作弊菜单

### 物品生成器使用

#### 方式一：快速生成（推荐）
1. 切换到「物品生成」标签页
2. 在搜索框输入物品名称或ID（如：`手枪` 或 `101`）
3. 点击「搜索」按钮
4. 在搜索结果中点击对应的按钮：
   - **x1**：生成1个
   - **x10**：生成10个
   - **x99**：生成99个
   - **x最大堆叠**：生成该物品的最大堆叠数量

#### 方式二：直接输入ID
1. 在「快速生成」区域输入物品ID
2. 输入生成数量
3. 点击「生成」按钮

### 玩家作弊功能

#### 无敌模式
1. 切换到「玩家作弊」标签页
2. 点击「切换无敌模式」按钮
3. 再次点击可关闭无敌模式

#### 快速传送
1. 切换到「玩家作弊」标签页
2. 点击「传送开关」按钮开启传送功能
3. 在游戏中，将鼠标指向想要传送到的位置
4. 按下 **鼠标中键（滚轮按下）** 即可传送
5. 再次点击「传送开关」可关闭传送功能

### 缓存管理

#### 首次使用
第一次使用时需要扫描游戏物品：
1. 切换到「物品生成」标签页
2. 点击「扫描游戏物品」按钮
3. 等待扫描完成（通常需要几秒钟）
4. 扫描完成后，物品列表会自动缓存

#### 更新缓存
如果游戏更新或添加了新物品：
1. 切换到「设置」标签页
2. 点击「重新扫描物品（更新缓存）」按钮
3. 等待扫描完成
