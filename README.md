# Duckov Cheat GUI Mod

![Version](https://img.shields.io/badge/version-0.1.0-blue.svg)
![Game](https://img.shields.io/badge/game-Escape%20from%20Duckov-green.svg)

## ✨ 功能特性

### 🎁 物品生成器
- **智能搜索**：支持按物品ID或名称搜索
- **批量生成**：一键生成1/10/99/最大堆叠数量的物品
- **详细信息**：显示物品价值、重量、堆叠上限等属性

### 👤 玩家作弊
- **无敌模式**：切换角色无敌状态
- **快速传送**：按下鼠标中键传送到鼠标指向的位置

### ⚙️ 设置管理
- **缓存管理**：重新扫描、删除缓存、打开缓存文件夹
- **调试工具**：输出物品列表到日志

## 📦 安装要求

### 前置条件
- 游戏：《Escape from Duckov》
- [BepInEx 5.x](https://github.com/BepInEx/BepInEx/releases)（Unity Mono版本）

### 依赖库
- BepInEx
- 0Harmony（包含在BepInEx中）
- Newtonsoft.Json（游戏自带）

## 🔧 安装步骤

### 1. 安装BepInEx
1. 下载适用于Unity Mono的BepInEx 5.x版本
2. 解压到游戏根目录（与游戏exe文件同级）
3. 运行一次游戏，让BepInEx初始化

### 2. 安装Mod
1. 将编译好的 `DuckovCheatGUI.dll` 文件放入：
   ```
   <游戏根目录>/BepInEx/plugins/
   ```
2. 确保文件路径正确，例如：
   ```
   steamapps\common\Escape from Duckov\BepInEx\plugins\DuckovCheatGUI.dll
   ```

### 3. 启动游戏
启动游戏后，在BepInEx控制台中应该能看到以下信息：
```
=================================
作弊菜单 Mod v0.1.0 已加载！
缓存文件: <路径>/DuckovCheatGUI_ItemCache.json
=================================
[成功] Harmony 补丁已应用
进入游戏后按 Home 键打开菜单
```

## 🎮 使用方法

### 打开/关闭菜单
- 按 **Home** 键打开或关闭作弊菜单

### 物品生成器使用

#### 方式一：快速生成（推荐）
1. 切换到「物品生成」标签页
2. 在搜索框输入物品名称或ID（如：`手枪` 或 `101`）
3. 点击「搜索」按钮
4. 在搜索结果中点击对应的按钮：
   - **x1**：生成1个
   - **x10**：生成10个
   - **x99**：生成99个
   - **x最大堆叠**：生成该物品的最大堆叠数量

#### 方式二：直接输入ID
1. 在「快速生成」区域输入物品ID
2. 输入生成数量
3. 点击「生成」按钮

### 玩家作弊功能

#### 无敌模式
1. 切换到「玩家作弊」标签页
2. 点击「切换无敌模式」按钮
3. 再次点击可关闭无敌模式

#### 快速传送
1. 切换到「玩家作弊」标签页
2. 点击「传送开关」按钮开启传送功能
3. 在游戏中，将鼠标指向想要传送到的位置
4. 按下 **鼠标中键（滚轮按下）** 即可传送
5. 再次点击「传送开关」可关闭传送功能

### 缓存管理

#### 首次使用
第一次使用时需要扫描游戏物品：
1. 切换到「物品生成」标签页
2. 点击「扫描游戏物品」按钮
3. 等待扫描完成（通常需要几秒钟）
4. 扫描完成后，物品列表会自动缓存

#### 更新缓存
如果游戏更新或添加了新物品：
1. 切换到「设置」标签页
2. 点击「重新扫描物品（更新缓存）」按钮
3. 等待扫描完成

#### 查看缓存
- 缓存文件位置：`<游戏目录>/BepInEx/config/DuckovCheatGUI_ItemCache.json`
- 点击「打开缓存文件夹」按钮可直接打开文件夹

## ⚠️ 注意事项

1. **必须进入游戏场景**：某些功能（如物品生成、传送）需要在游戏场景中才能使用，在主菜单中无法使用
2. **保存游戏前备份**：使用作弊功能可能影响游戏进度，建议使用前备份存档
3. **仅供单机使用**：本Mod仅适用于单机游戏，请勿在多人游戏中使用
4. **BepInEx控制台**：如遇问题，请查看BepInEx控制台的日志信息

## 🐛 常见问题

### Q: 按Home键没有反应？
**A**: 确保已经进入游戏场景（不是主菜单），并检查BepInEx控制台是否有错误信息。

### Q: 点击生成按钮后没有物品？
**A**: 
- 确认已经在游戏场景中（不是主菜单）
- 检查输入的物品ID是否正确
- 查看BepInEx控制台的日志信息

### Q: 扫描物品失败？
**A**: 
- 确保已经完全进入游戏场景
- 尝试在游戏中等待一段时间后再扫描
- 如果多次失败，请重启游戏后再试

### Q: 传送功能不工作？
**A**: 
- 确认已经开启传送开关（按钮显示「已开启」）
- 确保在游戏场景中
- 确认按的是鼠标中键（滚轮按下），不是滚动滚轮

### Q: 缓存文件在哪里？
**A**: 缓存文件位于 `<游戏目录>/BepInEx/config/DuckovCheatGUI_ItemCache.json`，可以在「设置」标签页点击「打开缓存文件夹」直接打开。

## 📝 开发说明

### 编译环境
- .NET Standard 2.1
- Visual Studio 2019 或更高版本 / Rider

### 编译步骤
1. 克隆或下载源代码
2. 修改 `DuckovCheatGUI.csproj` 中的 `GameDir` 路径为你的游戏目录
3. 打开解决方案并编译
4. 编译后的DLL文件位于 `bin/Debug/netstandard2.1/` 目录

### 项目结构
```
DuckovCheatGUI/
├── CheatGUI.cs              # 主要代码文件
├── DuckovCheatGUI.csproj    # 项目配置文件
├── DuckovCheatGUI.sln       # 解决方案文件
└── README.md                # 说明文档
```
